# WordleHelper
A script printing in the terminal existing word composed of possible letters.
To use it, launch the main.py the read the console.
This script print every word that respect those 3 conditions:
  - Contains exactly 5 letters (Can be change in the constant in lettersModifier.py)
  - Is composed of a letter in the letters list possible for the index
  - Belongs to the english dictionary (data.json)
